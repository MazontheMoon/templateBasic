Fonts can be styled as normal, as italic, or as oblique.

Font weight can be normal, bold, lighter, bolder, or can be fine tuned using numerical values from 100 - 900

Text transform options are none, uppercase, lowercase, capitalise, and full-width.

Text decoration options are none, underline, overline, underline and overline together, and line-through.
The text decoration line can be styled as solid, double, dotted, dashed and wavy.

Multiple properties can be set through shorthand in this order:
font-style font-variant font-weight font-stretch font-size line-height font-family

Text drop shadows have 4 properties:

  1. The horizontal offset
  2. The vertical offset
  3. The blur radius (default = 0)
  4. Base colour (default = black)

Think dictionary style list

list-style-type sets the type of bullets to use

  • circle
  • disc
  • square
  • none
  • default

Numbered lists can also be set

  1. decimal
  2. decimal leading zero
  3. lower alpha
  4. lower latin
  5. lower roman
  6. upper alpha
  7. upper latin
  8. upper roman
  9. default

list-style-position sets whether bullet appears inside the list items or outside and before each item

  • default
  • inside
  • outside

using 

list-style-image: url();

allows you to choose image to use as bullets - however is limited in terms of controlling position or size.
Instead can use the background family of properites (don't forget to set to no-repeat)

  • image
  • background-image
  • You can also insert "string" content to look like a bullet
ul { list-style-type: square; list-style-image: url(example.png); list-style-position: inside;}

could be written as

ul { square url(example.png) inside; }

Values can be listed in any order and can use 1, 2 , or all 3. Defaults not specified are disc none outside.

  2. 2
  3. 3
  4. 4
